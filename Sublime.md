### Hot Keys
`shift+RMouse` = vertical select

`Ctrl + D` - select all vars

`Ctrl + H` — Replace

`Ctrl + Shift + D` - Copy code line

`F9 и Ctrl+F9` — sort strings ASC (!HUOM! = int as string)
